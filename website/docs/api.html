Control API

Header required when configured: X-AgentSwarm-Token: <TOKEN>.

Read endpoints

  • GET /v1/status
  • GET /v1/peers
  • GET /v1/blocked
  • GET /v1/mcp/policy
  • GET /v1/inbox
  • GET /v1/inbox/unread

Write endpoints

  • POST /v1/connect
  • POST /v1/send
  • POST /v1/block
  • POST /v1/unblock
  • POST /v1/inbox/ack

Example

curl -X POST http://127.0.0.1:8787/v1/send \
  -H "X-AgentSwarm-Token: <TOKEN>" \
  -H "Content-Type: application/json" \
  -d '{"pubkey":"<hex>","type":"message","payload":{"text":"hello"}}'
